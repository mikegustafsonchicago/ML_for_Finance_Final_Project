KORD Random Forest: Wind Speed Prediction Analysis

Model Architecture

This analysis implements a multivariate time series regression model using Random Forest to predict wind speed at Chicago O'Hare International Airport (KORD).

Model Type

  • Base Model: Random Forest Regression
  • Feature Engineering: Time-lagged features for all numeric variables
  • Prediction Target: Next hour's wind speed

Feature Details

Feature Name Description Units Time Window
Wind Speed Current and historical wind speed measurements m/s 24-hour lag
Wind Direction Current and historical wind direction measurements degrees 24-hour lag
Wind Direction Delta Change in wind direction between consecutive hours degrees Current
Temperature Current and historical temperature measurements °C 24-hour lag
Humidity Current and historical humidity measurements % 24-hour lag
Dew Point Current and historical dew point measurements °C 24-hour lag
Sea Level Pressure Current and historical pressure measurements hPa 24-hour lag
Visibility Current and historical visibility measurements km 24-hour lag
Cloud Coverage Current and historical min/max cloud coverage ft 24-hour lag

Training Configuration

  • Train/Test Split: 80/20
  • Random State: 42 (for reproducibility)
  • Number of Trees: 100
  • Max Depth: 10
  • Min Samples Split: 5
  • Min Samples Leaf: 2

Model Interpretation

This Random Forest model captures complex non-linear relationships between various weather parameters and wind speed. The model:

  • Learns complex patterns through an ensemble of decision trees
  • Automatically handles feature interactions
  • Provides robust feature importance rankings
  • Is less prone to overfitting compared to single decision trees

Future improvements could include:

  • Hyperparameter tuning using grid search or random search
  • Feature selection based on importance scores
  • Ensemble methods combining multiple models

Model Performance

Metric Value
Root Mean Squared Error (RMSE) 4.54
Mean Absolute Error (MAE) 3.45
R² Score 0.70
