Wind Speed Prediction Analysis

Model Architecture

This analysis implements a multivariate time series regression model to predict wind speed at Chicago O'Hare International Airport (KORD).

Model Type

  • • Base Model: Multivariate Linear Regression
  • • Feature Engineering: Time-lagged features for all numeric variables
  • • Prediction Target: Next hour's wind speed

Feature Details

Feature Name Description Time Window
Wind Speed Current and historical wind speed measurements 24-hour lag
Wind Direction Current and historical wind direction measurements 24-hour lag
Wind Direction Delta Change in wind direction between consecutive hours Current
Temperature Current and historical temperature measurements 24-hour lag
Humidity Current and historical humidity measurements 24-hour lag
Dew Point Current and historical dew point measurements 24-hour lag
Sea Level Pressure Current and historical pressure measurements 24-hour lag
Visibility Current and historical visibility measurements 24-hour lag
Cloud Coverage Current and historical min/max cloud coverage 24-hour lag

Training Configuration

  • • Train/Test Split: 80/20
  • • Random State: 42 (for reproducibility)
  • • Validation: Standard train-test split

Model Interpretation

This multivariate regression model captures the complex relationships between various weather parameters and wind speed. The model:

  • Considers the impact of all available weather parameters
  • Accounts for changes in wind direction through the wind_dir_delta feature
  • Provides insights into which factors most influence wind speed

Future improvements could include:

  • Feature selection to reduce dimensionality
  • Non-linear models to capture complex relationships
  • Time series specific models (ARIMA, LSTM)

Model Performance

Metric Value
Root Mean Squared Error (RMSE) 5.72
