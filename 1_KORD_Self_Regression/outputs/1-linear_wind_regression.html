KORD Linear Regression: Wind Speed Prediction Analysis

Model Architecture

This analysis focuses on predicting wind speeds at Chicago O'Hare International Airport (KORD) using historical weather data.

Key Implementation Details

  • Time Window: 24-hour historical window to capture daily patterns
  • Wind Direction Handling: Special delta feature to handle circular wind measurements
  • Feature Engineering: Hourly lags of all weather parameters to capture temporal dependencies

Feature Details

Feature Name Description Units Time Window
Wind Speed Current and historical wind speed measurements m/s 24-hour lag
Wind Direction Current and historical wind direction measurements degrees 24-hour lag
Wind Direction Delta Change in wind direction between consecutive hours degrees Current
Temperature Current and historical temperature measurements °C 24-hour lag
Humidity Current and historical humidity measurements % 24-hour lag
Dew Point Current and historical dew point measurements °C 24-hour lag
Sea Level Pressure Current and historical pressure measurements hPa 24-hour lag
Visibility Current and historical visibility measurements km 24-hour lag
Cloud Coverage Current and historical min/max cloud coverage ft 24-hour lag

Training Configuration

  • Train/Test Split: 80/20
  • Random State: 42 (for reproducibility)
  • Validation: Standard train-test split

Model Interpretation

Key findings for wind prediction at KORD:

  • Wind direction changes provide significant predictive power
  • 24-hour historical window captures daily wind patterns
  • Pressure and temperature gradients show strong correlation with wind speed changes

Potential improvements specific to KORD:

  • Incorporate runway configuration data
  • Add seasonal interaction terms
  • Include Lake Michigan effect variables
