KORD Neural Network: Temperature Prediction Analysis

Model Architecture

This analysis implements deep neural network models to predict temperature at Chicago O'Hare International Airport (KORD) across multiple time horizons.

Model Type

  • Base Model: Deep Neural Network (PyTorch)
  • Feature Engineering: Time-lagged features for all numeric variables
  • Prediction Horizons: Multiple time windows (1h, 24h, 5d, 5d avg, 30d avg)

Feature Details

Feature Name Description Units Time Window
Temperature Current and historical temperature measurements °C 24-hour lag, ±2/5 day, 5-day avg
Wind Speed Current and historical wind speed measurements m/s 24-hour lag
Wind Direction Current and historical wind direction measurements degrees 24-hour lag
Wind Direction Delta Change in wind direction between consecutive hours degrees Current
Humidity Current and historical humidity measurements % 24-hour lag
Dew Point Current and historical dew point measurements °C 24-hour lag
Sea Level Pressure Current and historical pressure measurements hPa 24-hour lag
Visibility Current and historical visibility measurements km 24-hour lag
Cloud Coverage Current and historical min/max cloud coverage ft 24-hour lag

Training Configuration

  • Train/Test Split: 80/20
  • Random State: 42 (for reproducibility)
  • Network Architecture: 4-layer neural network (128→64→32→1)
  • Activation: ReLU
  • Dropout: 0.2
  • Optimizer: Adam (lr=0.001)
  • Batch Size: 32
  • Early Stopping: 10 epochs patience

Model Interpretation

These deep neural network models capture complex non-linear relationships between various weather parameters and temperature across different time horizons. The models:

  • Learns hierarchical features through multiple layers
  • Uses dropout for regularization to prevent overfitting
  • Implements early stopping to optimize training duration
  • Provides feature importance based on first layer weights

Future improvements could include:

  • LSTM layers to better capture temporal dependencies
  • Hyperparameter tuning using grid search or Bayesian optimization
  • Ensemble methods combining multiple neural network architectures
  • Attention mechanisms to focus on relevant time steps
